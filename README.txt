SMP Web App lista para publicar

Pasos:
1. Sube esta carpeta a GitHub o descomprímela en tu computadora.
2. Abre terminal dentro de la carpeta.
3. Ejecuta: npm install
4. Ejecuta: npm run dev
5. Para publicar en Vercel:
   - crea un repositorio en GitHub
   - sube estos archivos
   - en Vercel elige New Project e importa el repositorio
   - Deploy

Esta app usa:
- React + Vite
- jsPDF para generar cotizaciones
- localStorage para guardar proyectos en el navegador
